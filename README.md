# AquaFlow -- Swimming Pool Management System

AquaFlow is a **full‑stack Swimming Pool Management System** developed
using **Django** for the backend and **HTML, CSS, JavaScript, and
Tailwind CSS** for the frontend.\
This system helps manage swimming pool memberships, users, and trainers
through a dynamic and interactive web interface.

## Features

-   **User Registration & Login**
    -   Users can create an account and securely log in to the system.
-   **Trainer Login**
    -   Trainers have a separate login panel to manage their activities.
-   **User Dashboard**
    -   Each registered user gets a personalized dashboard.
-   **Trainer Dashboard**
    -   Each registered Trainer gets a personalized dashboard.
-   **Membership Plans**
    -   Users can view and purchase available swimming pool membership
        plans.

-   **Digital Pool Pass**
    -   Members can generate their **membership pass** for swimming pool
        access.
-   **Data Management**
    -   All user information, membership details, and trainer data are
        securely stored in the database.

## Technologies Used

-   **Backend:** Django (Python Web Framework)
-   **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
-   **Database:** MySQL
-   **Other Tools:** Django Templates, Static Files

## Project Purpose

The goal of this project is to simplify swimming pool management by
providing a digital platform where:

-   Users can easily register and manage their memberships
-   Trainers can access the system through their own login
-   Membership details and passes can be generated digitally
